OkoVision

Interface Web de supervision des chaudieres OeKofen Pellomatic Touch

View on GitHub Download .zip Download .tar.gz

Importer les fichiers générées par la chaudière manuellement ou automatiquement
et choissisez ce que vous voulez voir.

Pourquoi faire ?

Il n'est pas forcement evident de bien regler votre installation car chaque installation/maison est differente et unique
Pour optimiser votre installation, vous devre suivre cette methodologie

Les reglages ne sont pas facile si vous ne disposez pas de graphque vous permettant de suivre le fonctionnement de votre chaudiere au plus pret. C'est devant cette problematique que j'ai decidé de developper cette application.

Ainsi, il est maintenant possible de superviser une chaudière Oekofen grace à OkoVision

Pour qui ?

Pour tous les possesseurs de chaudieres oekofen Pellomatic Touch

Comment récupérer les données de la chaudière

Vous avez 2 choix possibles :

  • Par http, vous devez relier votre chaudiere a votre reseau informatique domestique (connecté à votre box)
  • Par USB, en branchant une clef usb sur votre chaudiere et d'uploader ensuite dans Okovision les fichiers générées

Tous cela est décrit dans le Wiki

Cela marche avec toutes les installations Pellomatic touch ?

Oui, lors de l'installation d'okovision, vous faites apprendre au programme comment est constitué votre installation
OkoVision s'adapte à toutes les installations.

Un synthèse à l'année est-elle possible ?

OkoVision vous offre une synthèse pour chaque mois et sur la saison.
